#Guia para compilar y ejecutar la aplicaicón Product Manager

Este documento proporciona una guía paso a paso para compilar y ejecutar la aplicación ProductManager utilizando el entorno de programación Android Studio.

## Requisitos Previos

- Tener instalado Android Studio
- Tener instalado Java 8

## Ejecución y compilación

Para ejecutar y compilar la aplicación de ProductManager en Android Studio se deberan seguir las siguientes instrucciónes:

1. Añadir un dispositivo virtual para emular la aplicación. (Tools --> Device Manager --> Create device)
2. Seleccionar un dispositivo con API > 26
3. Iniciar el entorno de simulación del dispositivo añadido.
4. Pulsar sobre el boton Play del menú superior.
